The Social Internet of Things

Join us in building decentralised social networks for the IoT today!

Decentralised

Hubber.space gives back control to communities. Anyone can deploy a server and connect Things to a social network. Your deployment has access to the whole ecosystem of apps and other networks.

Interoperability

An essential component of unlocking the IoT's potential is interoperability. Hubber.space integrates with PIOTRe and TritanDB to allow Things to join Hubber.space networks and support data and app semantic interoperability.

Fast Pub-Sub

Hubber.space uses a selective-publish disruptor pattern MQTT broker to disseminate information across the network, which performs very quickly in benchmarks even on resource-constrained servers.

Open Source and Cross Platform

Hubber.space is licensed under the Apache License 2.0, which means its totally free and open source! It is built with Node.js and runs across platforms and cloud computing services.

Learn more, see the source code or dive in and get started!

Create your own network

You can create your own decentralised hubber.space network by deploying the hubber.space app on your own server. Download hubber.space for free on Github or if you're adventurous you can try building it from source.
